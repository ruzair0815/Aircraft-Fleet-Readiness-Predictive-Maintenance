# Aircraft Fleet Readiness & Predictive Maintenance
Predictive maintenance analytics project using aircraft engine sensor data to estimate Remaining Useful Life (RUL), identify critical engines, and visualize fleet health trends through an interactive Tableau dashboard.

## Features
- Analyze aircraft engine sensor data from multiple operating cycles.
- Estimate Remaining Useful Life (RUL) for each engine.
- Identify critical engines approaching failure thresholds.
- Clean and transform large sensor datasets using Python and Pandas.
- Create KPI metrics and interactive Tableau dashboards for fleet monitoring.
- Visualize engine degradation trends, heatmaps, and maintenance risk insights.

## Technologies
- Backend/Data Processing: Python, Pandas, NumPy
- Data Visualization: Tableau
